## ✨ Features

* 🐰 Adorable kawaii bunny character
* ⭐ Catch stars for points
* 🌈 Rare rainbow stars worth bonus points
* 🔥 Combo multiplier system up to x5
* 🛡 Shield power-up
* 🧲 Magnet power-up
* ⏳ Slow-time power-up
* ☁️ Dangerous storm clouds
* ❤️ Three-life survival system
* 📈 Progressive difficulty across 5 levels
* 📱 Mobile and touch support
* 🎨 Elegant pastel aesthetic

## 🎮 How to Play

* Move your mouse (or finger on mobile) to control the bunny basket.
* Catch stars to earn points.
* Avoid storm clouds.
* Collect power-ups.
* Build combos for higher scores.
* Survive until time runs out.
